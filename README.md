# KeyGuard Password Manager
### Console based(CLI) Password Manager written in Python 3
```
Features:

1. Minimal
2. Encrypted
3. Multi-User
4. Lightweight and Fast
5. Intuitive UI
```
Download Through Git:
```
git clone https://github.com/0xViKi/keyguard-password-manager.git
```

To run:
```python3
python3 manager.py
```
